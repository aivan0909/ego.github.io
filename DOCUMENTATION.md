# 專案開發導覽 (Project Walkthrough) - One Sentence a Day

這個文件記錄了本專案的核心功能發展、語法邏輯以及開發過程中的重要修復。

## 🚀 核心功能優化

### 1. 智慧語法組合系統 (SVO & FULL)
- **單複數自動匹配 (Singular/Plural Agreement)**：
    - 主語 (Subject) 現在帶有 `isPlural` 屬性。
    - 當手動選擇或「智慧隨機組合」時，動詞 (Verb) 會自動過濾，確保與主語的單複數形式一致（例如：*The boy likes* vs *They like*）。
    - 顏色樣式同步：SVO 與 FULL 按鈕採用與「智慧隨機組合」相同的藍紫色漸層配色。

### 2. 每日短文擴充 (Daily News)
- **內容擴量**：由原本的 3 篇擴充至 **20 篇精選文章**，主題涵蓋科技 (TSMC AI)、在地生活 (台灣夜市、天氣)、健康與探索。
- **雙重發音系統**：
    - **全文朗讀**：點擊文章標題旁的發音按鈕可朗讀整篇內容。
    - **分句點選發音**：點擊內文中的任何一個英文句子，系統會單獨朗讀該語句，方便細緻練習。
- **視覺優化**：發音圖示更新為彩色高品質樣式，佈局採用英文與中文對照排列。

### 3. 詞彙量大幅擴增
- **新增 200 組相關字詞**：針對兒童 (child)、青少年 (teen)、成人 (adult) 不同情境新增了大量主語、動詞、賓語及修飾詞。
- **跨年齡層翻譯優化**：當在當前年齡層找不到對應翻譯時，系統會自動搜尋其他分類的詞庫，確保中文翻譯不留白。

## 🛠️ GitHub Pages 部署修復紀錄
在開發過程中，GitHub Pages 曾出現空白畫面，我們已透過以下方式解決：
- **路徑重定向**：將 Vite 的 `base` 設定為 `/ego.github.io/`，確保在 GitHub 子路徑下能正確找到資產。
- **資源檔名鎖定**：取消 Vite 打包時的檔名亂碼（Hash），強制固定為 `index.js` 與 `index.css`，避開伺服器快取問題。
- **部署分支**：確定將打包後的 `dist` 內容推送至 `gh-pages` 分支。
